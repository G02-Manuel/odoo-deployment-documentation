# Odoo Documentation

Documentacion para la asistencia en el proceso de instalacion de Odoo Open ERP en un servidor linux independientemente del propietario del mismo.

El objetivo de la presente documentacion es definir el proceso de instalacion y configuracion de odoo en version 12.0, incluyendo todo lo necesario para realizar el despliegue del sistema, administracion del servicio, administracion del servicio de base de datos y configuracion del ambiente con los modulos de la comunidad.
